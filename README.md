# Dotfiles
Dotfiles are files with a dot infront of them (`.`). On Linux, this is a way to signify the file should be hidden from the standard viewer. Often times these files can still be accesed but you have to know they are there! _(`ls -a` can help with this)_

Most configuration files are not needed for the average viewer. You set up your configuration once and you never look at them again. However, if you ever switch computers or workspaces, it is nice to still have the same configuration for programs such as `nvim` and `tmux`. 

Hence we put them on our github for all to see!
